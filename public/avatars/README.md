# Avatar Assets

## Structure

This directory should contain 18 avatar images:

- **Male avatars**: `1.jpg` to `10.jpg` (10 avatars)
- **Female avatars**: `11.jpg` to `18.jpg` (8 avatars)

## Specifications

- Format: `.jpg`
- Recommended size: 400x400 pixels minimum
- Aspect ratio: 1:1 (square)
- File size: < 200KB per image

## Usage

Avatars are referenced in the app using the format `avatar_{id}`, where `{id}` is the avatar number (1-18).

Example:
- `avatar_1` → `assets/avatars/1.jpg`
- `avatar_15` → `assets/avatars/15.jpg`

## Note

Please ensure all 18 avatar images are added to this directory before building the app for production.
